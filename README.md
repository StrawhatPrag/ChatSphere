# ChatSphere

A real-time multi-client chat application built using Java sockets and JavaFX.

## Features

- Real-time messaging
- Multiple client support
- Online user tracking
- Private messaging
- JavaFX GUI
- Message bubbles
- User avatars

## Tech Stack

Java
JavaFX
Socket Programming
Multi-threading
Maven

## Architecture

Client-Server architecture.

Server manages connections and broadcasts messages.

Clients communicate through TCP sockets.
